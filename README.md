# Raman-database---biological-spectra
Interactive database to search literature on Raman peaks assigned to biological structures

Use this tool as a literature database of Raman peaks. The listed wavenumbers are commonly found in biological tissues, cells and cell cultures. 
To use the interactive database:
  * Download the file 'test_v6.html'
  * Open in your browser of choice (Google Chrome works guaranteed)
  * Ensure an internet connection

The database is now ready to use!
